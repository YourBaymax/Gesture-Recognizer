Gesture-Recognizer
基于win10＋python3.5
安装tensorflow，最好安装gpu版。win10上跑CNN比较慢，如果不用gpu版的tensorflow速度会更慢。
安装keras，theano（这个在程序中可能用不到）
安装opencv和一些乱七八糟的库，缺什么就安装什么。
